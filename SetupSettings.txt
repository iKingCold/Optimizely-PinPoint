SearchPrefix
Leave empty for no prefix (Free search), ex:
SearchPrefix: []
Enter one prefix for static prefix, ex:
SearchPrefix: [Stockholm]
Enter multiple prefixes separated by comma for a dropdown-list, ex:
"SearchPrefix": [ "Stockholm", "Skövde" ],
Can also enter empty string in array to allow editors to free-search with the dropdown, ex:
"SearchPrefix": ["Stockholm, Skövde", ""]

Identifier: 
Specifies the tile-theme, ex: "topowebb" or "topowebb_nedtonad"
